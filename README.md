# LuxLink2-QE-Simulator
Building a bridge between hardware and software, LuxLink2 introduces a real-time data pipeline. Using Python and pyserial, the system streams every "spooky" collapse event from the Arduino to a digital dashboard. This repo focuses on asynchronous data logging and visualising the transition from physical superposition to digital certainty.
